Dotfiles repository
====

## Overview
.vimrc .bashrc .tmux.conf .dir_corors etc

## Usage

#### proxy
    git config --global http.proxy http://xxx:xxx
    git config --global https.proxy http://xxx:xxx

#### clone
    git clone http://github.com/kanon07/dotfiles.git

#### Install
    cd dotfiles
    ./setup.sh

#### without proxy
    git checkout noproxy

    ./setup.sh
